# Tools Directory

This directory contains self-developed tools that are worth keeping in this repository.

## Categories

- `automation`: workflow automation and scheduled helpers
- `data`: data extraction, conversion, validation, and report helpers
- `dev`: developer workflow and engineering utilities
- `file`: file operations, packaging, and synchronization helpers
- `media`: image, audio, video, or document processing
- `system`: environment, process, and OS level utilities
- `text`: text processing, parsing, templating, and prompt helpers

Each tool should live in `tools/<category>/<tool-id>/`.
